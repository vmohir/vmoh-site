# Chooser

A Chwazi-style finger picker for mobile. Put two or more fingers on the screen,
hold for a moment, and the app:

- **One winner** — picks a single random finger.
- **Split teams** — assigns each finger to one of 2–6 teams.
- **Order** — gives each finger a 1, 2, 3… position.

Works offline. Installable to the home screen on iOS and Android.

## Develop

```
pnpm install
pnpm dev          # http://localhost:4321
pnpm build        # static build → dist/
pnpm preview
```

See `CLAUDE.md` for architecture and conventions.
